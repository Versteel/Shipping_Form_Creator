# Introduction

Welcome to the Shipping Form Creator, a desktop application designed to streamline the process of generating, managing, and printing critical shipping documents, including Packing Lists and Bills of Lading (BOL).

This application is built for internal use at Versteel, connecting directly to the company's ERP system to pull real-time order data. It provides a user-friendly interface to organize shipment details, group items into handling units, manage multi-truck shipments, and print standardized, professional-looking documents.

## Key Features

*   **Real-time Data Integration:** Fetches live sales order information from the ERP system.
*   **Packing Lists:** Generate detailed packing lists showing all items in a shipment.
*   **Bills of Lading:** Create legally compliant Bills of Lading with automated calculations for freight class and NMFC codes.
*   **Handling Units:** Group individual packing units (boxes, skids) into larger handling units (pallets) using a simple drag-and-drop interface.
*   **Multi-Truck Shipments:** Assign items to different trucks and view the contents of each truck separately.
*   **Local Caching:** User modifications are saved locally, so your work is preserved and automatically merged with fresh data from the ERP.
*   **Search Functionality:** Easily find past and present orders by order number or ship date.
*   **Printing:** Print standardized documents directly from the application.

This documentation will guide you through the features and usage of the Shipping Form Creator.
