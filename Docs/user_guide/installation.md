# Installation

The Shipping Form Creator is a desktop application distributed internally at Versteel. Please follow the instructions below based on your situation.

## First-Time Installation

If you do not have the application installed, please contact the IT department. They will provide you with the installer package or a link to a network share where you can download the application.

Run the provided installer and follow the on-screen prompts to complete the installation.

## Application Updates

The Shipping Form Creator is periodically updated to include new features and bug fixes. The method for updating the application will depend on how it was initially deployed.

*   **Automatic Updates:** In some cases, the application may automatically check for and install updates upon startup.
*   **Manual Updates:** If you are notified that a new version is available, you may need to re-run the installer package provided by the IT department.

If you have any questions about installation or updates, please submit a help desk ticket to the IT department for assistance.
